01 — Overview

An AI inspector for the production line

Manual visual inspection of printed circuit boards is slow, inconsistent, and fatiguing. This project trains a YOLOv12 detector on an augmented PCB dataset and wraps it in a simulated AOI station: test boards are streamed through the model one by one, producing live PASS / FAIL verdicts, a defect-class distribution, a yield trend, and an SPC-style alarm that fires after three consecutive failures — exactly the signals a real inline inspection cell would surface.

  • ⚡
    Real-time detection
    ~15 ms GPU inference per 512×512 board.
  • 🎯
    High precision, low false alarms
    95.1% precision keeps good boards moving.
  • 📈
    Process feedback
    Yield trend & consecutive-fail alarm for SPC.

06 — Key Findings

What we learned

✓

YOLOv12 reaches stable convergence without overfitting across 100 epochs.

✓

Augmentation (brightness / contrast / flip / rotation) effectively simulates production-line variance in lighting and orientation.

✓

Suitable for real-time quality inspection with fewer false alarms on reflective or rotated boards.

✓

Demonstrates an efficient, reproducible pipeline for automated defect detection in manufacturing.
